## Упражнения

### ex1_neuron — нейрон и нелинейность

Реализуете `relu`, `neuron` (взвешенная сумма + активация) и линейный
нейрон без активации. Итоговая демонстрация: **без функции активации два
слоя сворачиваются в один линейный**, а с ReLU — нет (§3 «Почему без неё
сеть не работает»).

```bash
uv run python 01-foundations/ex1_neuron/work.py
```

### ex2_training_loop — цикл обучения вручную

Собираете четыре шага обучения из §5 (forward → loss → backprop → шаг
оптимизатора) для линейной регрессии `y = w*x + b`, вычисляя градиенты MSE
**руками** (без autograd). Вторая часть решения демонстрирует **overfitting**:
полином степени 5 идеально ложится на 6 точек, но плохо обобщает.

```bash
uv run python 01-foundations/ex2_training_loop/work.py
```

> Связь с 05: тот же цикл обучения, но на реальном PyTorch с `autograd` и
> `optimizer`, разбирается в [05/ex3_pytorch_training](../05-pipelines-frameworks/README.md).
