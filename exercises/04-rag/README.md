## Упражнение

### ex1_mini_rag_pipeline — мини-RAG

Собираете полный RAG-пайплайн из §5–§6: эмбеддинг документов → поиск top-K
по `cos_sim` → сборка промпта «отвечай только по контексту» → генерация.

База знаний содержит **вымышленный факт** («Зимородок-7 сокращает
латентность на 42%»), которого нет в обучающих данных модели. Решение
показывает разницу: **без RAG** модель факт не знает, **с RAG** —
отвечает точно. Это и есть главная польза RAG (§1–§2 теории).

```bash
uv run python 04-rag/ex1_mini_rag_pipeline/work.py
```

> Переиспользуется в практике 08 (`ex2_prompt_injection`): тот же mini-RAG
> используется для демонстрации косвенной инъекции через документ.

> Как запускать упражнения (`setup_practice.py`, `work.py`, `solution.py`) —
> см. [общий README практики](../README.md#3-конвенция-файлов).
